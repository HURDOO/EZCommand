EZCommand
==========
블록 코딩, 마인크래프트 커맨드 생성기

## 소개
EZCommand는 마인크래프트 커맨드 구문 작성을 더욱 수월하게 만들어주는 프로그램입니다. 구문 작성이 어려운 분들을 위한 블록 코딩 기능과, 명령어 문법 하이라이트(추가 예정) 등등을 지원합니다!
